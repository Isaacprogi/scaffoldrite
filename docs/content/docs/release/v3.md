# 🚀 Scaffoldrite v3

Release Date: 2026-03-15

## ✨ New Features
- `deps` command: Detect circular dependencies & standalone files an serve file relation.
- `lock/unlock` commands: Lock CI, git hooks, or structure
- Enhanced `generate` command with ignore-tooling support
- Improved `find` command to search both structure & filesystem
* Added `--against` flag to run commands against Git references
* Added Git-based filesystem generation
* Added dependency graph command (`deps`)
* Added CI structure locking

## ⚠️ Improvements
- Better validation messages
- Improved progress output
