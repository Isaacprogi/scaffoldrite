Scaffoldrite is built **by developers, for developers**. Whether you’re a solo founder, a team lead, an open source maintainer, or a freelancer, we want you to be part of our community.

---

## 🌐 Join the Community

- **GitHub Repository:** [⭐ Star & Contribute](https://github.com/Isaacprogi/scaffoldrite)  
- **Issues:** [🐛 Report bugs](https://github.com/Isaacprogi/scaffoldrite/issues)  
- **Discussions:** [💬 Share ideas & ask questions](https://github.com/Isaacprogi/scaffoldrite/discussions)  

**Tips for contributing:**
1. Check existing issues before creating a new one  
2. Follow the coding & structure conventions in `.scaffoldrite/structure.sr`  
3. Keep pull requests small & focused  
4. Add tests where possible, especially for generators and validators  

---

## 💡 Best Practices for Contributors

- **Respect structure:** Scaffoldrite enforces consistency, so contributions should follow the defined patterns  
- **Document your changes:** Update `README.md` or docs/ files when adding features  
- **Test locally:** Use `sr generate` and `sr validate` to ensure no structural regressions  
- **Engage in discussions:** Help newcomers and maintainers by sharing best practices
