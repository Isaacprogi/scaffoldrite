# Scaffoldrite File Relation

Visualize, explore, and understand your project architecture with interactive dependency graphs.

## Features

* Interactive dependency graph visualization
* File and folder relationship mapping
* Dependency tracing
* Circular dependency detection
* Fast project exploration
* Search and filter support
* Modern and responsive UI

## Built With

* React
* TypeScript
* Tailwind CSS
* Graph visualization libraries
